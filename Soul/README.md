# Soul — 中国体育赛事聚合系统

> **提示词:** 设计中国体育赛事聚合系统方案，聚焦中国队比赛，支持多源免费直播数据抓取与融合，提供订阅推送机制

## 项目简介

聚焦 **中国队比赛** 的体育赛事聚合平台，自动抓取 12+ 免费数据源（央视 EPG、直播吧、FIFA、咪咕等）的赛程与直播链接，融合去重后提供统一视图，支持用户按运动/球队订阅推送。

## 核心功能

- **多源赛程抓取** — 插件化抓取器，覆盖央视 EPG API、直播吧、FIFA/FIBA/BWF/ITTF 官网、咪咕视频等
- **智能数据融合** — 六阶段流水线（归一化→匹配→决策→充实→冲突解决→变更检测），自动去重合并
- **队名归一化** — 别名表 + rapidfuzz 模糊匹配（"国足"/"中国男足"/"China PR" → 同一实体）
- **订阅推送** — 赛前 24h/1h 提醒、开赛直播链接推送、实时比分、赛后结果，多通道分发
- **直播流聚合** — 自动发现 CCTV5、咪咕、央视频等免费直播链接

## 技术栈

| 层级 | 技术 |
|------|------|
| API 框架 | FastAPI (async) |
| 数据库 | PostgreSQL 16 + Redis 7 |
| 抓取引擎 | httpx + Playwright |
| 任务调度 | APScheduler + Celery |
| 推送通道 | Telegram Bot / 企微 Webhook / 微信服务号 / Email |
| 部署 | Docker Compose |

## 覆盖运动

足球、篮球、排球、乒乓球、羽毛球、游泳、田径、综合赛事（奥运会/亚运会）等
