# 始皇顶级音乐播放器 | Neo Music Helper

## 提示词

> 帮我制作一个顶级音乐播放器：
> 1. 样式高级，可切换风格
> 2. 有始皇跳舞动态效果（支持切换拉丁舞，街舞，中国特色民族舞）
> 3. 多种功能支持
>
> 名称：始皇顶级音乐播放器（neo music helper）

**模型：** Claude Opus

## 使用方式

直接用浏览器打开 `neo-music-helper.html` 即可使用，无需安装任何依赖。

## 功能

**6 大主题风格切换**
- 帝王金 / 赛博霓虹 / 暗夜星空 / 矩阵代码 / 樱花物语 / 深海幽蓝

**始皇跳舞动态效果**
- SVG 秦始皇角色，播放时随音乐节拍起舞
- 5 种舞蹈风格：默认、嘻哈、迪斯科、机器人、波浪
- 点击始皇触发彩蛋

**音频可视化**
- 4 种频谱样式（环形 / 条形 / 波形 / 粒子）
- 实时粒子特效 + 背景脉冲光效

**播放器功能**
- 拖拽或点击添加本地音乐文件
- 播放 / 暂停 / 上一首 / 下一首
- 随机播放 / 循环模式（关闭 / 列表循环 / 单曲循环）
- 音量控制 / 播放速度（0.5x ~ 2.0x）
- 播放列表管理
- 完整快捷键支持（按 `?` 查看）

## 技术栈

纯 HTML + CSS + JavaScript，零依赖，单文件，开箱即用。

- Web Audio API — 实时音频分析
- Canvas API — 频谱可视化 + 粒子特效
- SVG + CSS Animations — 始皇角色动画
