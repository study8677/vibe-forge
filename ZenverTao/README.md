# 愤怒的小鸟3D

> **提示词:** 用Three.js做一个3D版愤怒的小鸟网页游戏

## 项目简介

一款基于 Three.js 和 Cannon-es 物理引擎的 3D 版愤怒的小鸟网页游戏。玩家通过弹弓发射不同能力的小鸟，摧毁猪猪的防御工事，包含多关卡、星级评分和粒子特效系统。

## 核心功能

- 3D 弹弓物理投射系统，支持拖拽瞄准与释放发射
- 多种小鸟类型，各具独特技能（加速、分裂、爆炸等）
- 刚体物理引擎驱动的建筑破坏与碰撞检测
- 多关卡递进设计，含星级评分与过关判定
- 粒子特效系统（爆炸、烟雾、星星）

## 技术栈

| 层级 | 技术 |
|------|------|
| 3D 渲染 | Three.js 0.160 |
| 物理引擎 | Cannon-es 0.20 |
| 前端 | 纯 HTML/CSS/JS (ES Modules) |

## 文件说明

- `angry-birds-3d/index.html` — 游戏完整源码，单文件包含所有逻辑、样式与 3D 场景

## 运行方式

直接在浏览器中打开 `angry-birds-3d/index.html` 即可体验（需联网加载 Three.js 和 Cannon-es CDN 资源）。
