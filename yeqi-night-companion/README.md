# 夜气

> **提示词:** 做会读空气的深夜陪伴App：克制暧昧，识别情绪，主动陪伴，生成高保真交互原型。

## 项目简介

这是一个移动端优先的深夜陪伴 App 高保真交互原型，重点验证“先读空气，再主动靠近”的体验，而不是把产品做成泛聊天工具或虚拟恋人。原型通过三条夜间剧情线、低压力对话、呼吸干预和柔和收口，呈现一种有温度但不越界的陪伴感。

## 核心功能

- 深夜欢迎页与校准流程，设置陪伴距离、支持方式和暧昧边界
- “今晚首页”气氛判断视图，直接展示系统感知到的情绪空气
- 三条可切换的夜间剧情线：下班后空掉、闹别扭后强撑、凌晨失眠反刍
- 低压对话和身体优先的呼吸干预，支持从聊天自然切到放松模式
- 收口页总结今晚的情绪状态和陪伴方式，避免打卡化和运营化

| 层级 | 技术 |

| 界面 | Vanilla HTML + CSS |

## 文件说明

- `index.html` — 原型入口与页面壳
- `app/styles.css` — 夜色视觉系统、版式和动效
- `app/content.js` — 剧情线、文案、校准选项和收口内容
- `app/state.js` — 场景状态机、情绪策略映射和流程推进
- `app/app.js` — DOM 渲染和点击交互逻辑
- `tests/state.test.js` — 状态切换、剧情重置和策略映射测试
- `docs/superpowers/specs/2026-04-08-night-companion-design.md` — 产品设计文档
- `docs/superpowers/plans/2026-04-08-night-companion-prototype.md` — 实现计划

## 运行方式

1. 在项目目录执行 `npm run dev`
2. 打开 `http://127.0.0.1:4321`
3. 从欢迎页进入后，依次体验校准、今晚首页、对话、呼吸和收口流程
4. 如需验证状态逻辑，执行 `npm test`
