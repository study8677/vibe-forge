# 日升日落时间卡片

> **提示词:** 完成一个前端项目，语言不限，内容设计时间卡片翻转效果，背面同步现实时间展示太阳轨迹，实现日升日落与动态昼夜光影。

## 项目简介

一张可翻转的 3D 时间卡片。正面是经典分页翻牌式数字时钟，点击卡片翻转到背面，实时展示基于当前时间的太阳运行轨迹与动态昼夜天空光影变化。

## 核心功能

- 卡片 CSS 3D 翻转交互，正面时钟 / 背面太阳轨迹一键切换
- 分页翻牌式时钟，每位数字独立 3D 翻页动画
- Canvas 实时绘制太阳弧线轨迹，太阳位置与当前时间同步
- 天空渐变随时间连续过渡：深夜 / 黎明 / 日出 / 白昼 / 日落 / 黄昏
- 丰富场景元素：星空闪烁、月亮环形山、飞鸟编队、漂移云层、山脉剪影、湖面反射、萤火虫

## 技术栈

| 层级 | 技术 |
|------|------|
| 结构 | HTML5 |
| 样式 | CSS3（3D Transforms / Perspective / Keyframe Animations） |
| 渲染 | Canvas 2D（天空 / 太阳 / 星月 / 地形 / 水面） |
| 逻辑 | Vanilla JavaScript（太阳高度角计算 / 翻牌动画 / requestAnimationFrame） |

## 文件说明

- `index.html` — 单文件完整应用，包含全部 HTML / CSS / JS，零依赖

## 运行方式

直接用浏览器打开 `index.html` 即可体验，无需安装任何依赖。
