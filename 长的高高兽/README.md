# 亚马逊上架审查工作台

> **提示词:** 面向亚马逊跨境电商运营审查场景的桌面工作台，集中处理批量上架风险筛查与单图快速核查。

## 项目简介

面向亚马逊跨境电商运营团队的桌面审查工作台，集中处理两大高频场景：批量上架前的 Listing 风险筛查，以及单张产品主图的合规快速核查。上传 Excel/CSV 即可自动扫描标题违规、品牌侵权、价格异常等 8 个维度；拖入产品图则通过 Canvas 像素级分析完成纯白底、分辨率、产品占比等 8 项检测。

## 核心功能

- **批量风险筛查** — 上传商品表格，自动检测标题违规词、品牌侵权、价格异常、受限类目、搜索词 ASIN、描述外链等，按风险等级分级展示
- **单图合规检测** — 拖拽产品图，Canvas 像素级分析纯白底、分辨率、宽高比、文件大小、产品占比、边框检测，逐项给出通过/未通过与整改建议
- **风险评分体系** — 高/中/低三级风险自动评分，统计卡片 + 饼图总览，支持按风险等级筛选
- **逐条审核操作** — 通过/驳回/查看详情，侧抽屉展示按字段分组的风险明细与修改建议
- **报告导出** — 一键导出风险审查 Excel 报告，支持下载上架模板

## 技术栈

| 层级 | 技术 |
|------|------|
| 构建 | Vite 5 |
| 前端框架 | React 18 + TypeScript |
| UI 组件 | Ant Design 5 |
| 样式 | TailwindCSS 3 |
| 状态管理 | Zustand |
| 图表 | Recharts |
| 表格解析 | SheetJS (xlsx) |
| 图片分析 | Canvas API 像素级检测 |

## 文件说明

- `src/utils/riskRules.ts` — 亚马逊 Listing 风险规则引擎（标题、品牌、价格、类目、五点描述、搜索词、描述、SKU 共 8 维度）
- `src/utils/imageAnalysis.ts` — 产品图 Canvas 像素级合规分析（纯白底、分辨率、宽高比、文件大小、产品占比、色彩有效性、边框检测共 8 项）
- `src/utils/excelParser.ts` — Excel/CSV 解析、风险报告导出、模板生成
- `src/components/BatchReview/` — 批量筛查页组件（上传、统计仪表板、数据表格、风险详情抽屉）
- `src/components/ImageReview/` — 单图核查页组件（拖拽上传区、分析结果面板）
- `src/pages/` — 页面入口（BatchScreen、ImageCheck）

## 运行方式

```bash
npm install
npm run dev
```

浏览器打开 `http://localhost:5173`，点击「加载演示数据」可立即体验批量筛查效果。
